## About Empresa - FRONT-END

Bem-vindo ao Empresas, aplicativo desenhado para functionar Offline First, fácil e inteligente. <br>
Com a interface intuitiva, você pode criar, editar, visualizar e muito mais, tudo online ou offline. <br>
OBS: O projeto Empresas foi criado como forma de apresentação.

## Tecnologias 

React Native / TypeScript / JavaScript / Watermelondb / Android

## Instalação

1° Clonar o projeto <br>
2° cd empresa_aplicativo <br>
3° npm install <br>
4° npm run dev ou yarn dev <br>
5° OBS: Esse projeto contém apenas o Front-End do sistema <br>
   o Back-end se encontra em outro repositório <br>
   para rodar esse projeto necessariamente precisará do Back-end.<br>

## Como testar o Offline First

1° - Na tela inicial do app cadastre as empresas e usuarios. <br>

2° - Logue com um dos usuários criados. <br>

3° - Crie Lançamentos online ou offline - Sincronize com o banco API, <br>
Crie Empresas online ou offline - Sincronize com o banco API, <br>
Crie Usuários online ou offline - Sincronize com o banco API <br>
